compaitences

power bi

Power BI est un outil d'analyse commerciale développé par Microsoft qui permet de visualiser et de partager des informations issues des données de votre organisation.

excel

Microsoft Excel est un logiciel de tableur largement utilisé qui permet aux utilisateurs d'organiser, d'analyser et de visualiser des données.

recherche opérationelle

La recherche opérationnelle (RO) est une discipline qui utilise des méthodes analytiques et mathématiques pour prendre des décisions optimales dans des situations complexes.

statisctique mathématique

La statistique mathématique est une branche des mathématiques qui se concentre sur le développement et l'application de méthodes statistiques pour analyser des données.

html

HTML, est le langage de balisage standard utilisé pour créer et concevoir des pages web. Il s'agit d'une technologie fondamentale du World Wide Web qui permet de structurer le contenu des pages web.

css

CSS, ou "Cascading Style Sheets" (Feuilles de style en cascade), est un langage de style utilisé pour décrire la présentation d'un document HTML

javascript

JavaScript (JS) est un langage de programmation de haut niveau, orienté objet, principalement utilisé pour rendre les pages web interactives.

python

Python est un langage de programmation de haut niveau, interprété, et polyvalent. Il a été créé dans les années 1980 par Guido van Rossum et est devenu l'un des langages de programmation les plus populaires et largement utilisés.

cplex

CPLEX, ou IBM ILOG CPLEX Optimization Studio, est un ensemble de logiciels puissants dédiés à la résolution de problèmes d'optimisation mathématique.

c++

C++ est un langage de programmation généraliste, orienté objet, créé comme une extension du langage C. Développé par Bjarne Stroustrup au début des années 1980, C++ combine les fonctionnalités du langage C avec des fonctionnalités de programmation orientée objet, ce qui lui confère une grande flexibilité et une performance élevée.

msproject

Microsoft Project, souvent appelé MS Project, est un logiciel de gestion de projets développé par Microsoft. Il fait partie de la suite Microsoft Office et est conçu pour aider les chefs de projet à planifier, suivre et gérer les projets de manière efficace.

mysql

MySQL est un système de gestion de base de données relationnelle (SGBDR) open-source. Il est largement utilisé dans le développement web pour stocker et gérer des données.
